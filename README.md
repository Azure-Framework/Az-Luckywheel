# Az-Luckywheel

Lightweight, lore-friendly slot machines for the Diamond Casino. Auto-detects casino slot
props, spins real reel models, evaluates wins server-side, and shows a custom **Diamond
Casino–style win/loss/jackpot notification UI**.

**Resource:** `Az-Luckywheel`
**fx_version:** `cerulean`
**NUI page:** `html/index.html`

## What it does
Lightweight, lore-friendly slot machines for the Diamond Casino. Auto-detects casino slot
props, spins real reel models, evaluates wins server-side, and shows a custom **Diamond
Casino–style win/loss/jackpot notification UI**.

## Features
- Auto-detects Diamond Casino slot machines
- Finds world objects using the models listed in `config.lua`.
- Immersive interaction
- `E` near a machine to sit.
- `SPACE` to spin.
- `BACKSPACE` to stand up.
- Real reel props
- Spawns the correct reel prop per machine (e.g. `vw_prop_casino_slot_02a_reels`).
